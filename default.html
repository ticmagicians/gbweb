TEAM GREEN_BERETS 
Demo: CI/CD setup using Jenkins Pipeline Job to publish ASP website package to IIS server hosted on AWS EC2 Instances,  We will be demonstrating Blue-Green deployments by taking one of the EC2 instances out of the Loadbalance and deploying the new webpage updates and testing it before adding the instance back to the load balancer.  We are leveraging S3 buckets for publishing static web contents like images.
